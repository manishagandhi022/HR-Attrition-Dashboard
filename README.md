HR Attrition Analysis Dashboard using Power BI and SQL. Analyzed employee attrition trends by age, department, salary, and job role to identify key factors affecting employee turnover.

# HR Attrition Analysis
## Project Overview

This project focuses on analyzing employee attrition data to identify the key factors contributing to employee turnover. The analysis helps understand attrition trends based on department, age group, salary, job role, and other employee-related factors.

## Objective
Analyze employee attrition trends
Identify major factors affecting employee turnover
Track attrition across departments and job roles
Build an interactive dashboard for better insights
## Tools Used
SQL – data cleaning and analysis
Power BI – dashboard creation and visualization
Excel/CSV – dataset handling (agar use ki thi)
## Key Insights
Attrition rate across departments
Age groups with highest attrition
Salary and attrition relationship
Job roles with maximum employee turnover
Overall employee distribution and attrition trends
